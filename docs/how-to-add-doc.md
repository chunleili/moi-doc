# 创建文档的用法

想要创建一个新的文档页，非常简单，只需要三步。

1. 把这个moi-doc这个repo 拉下来
2. 新增文档的话，只要在docs目录下新增一个markdown文件即可。新增目录的话，只需要在docs下新建文件夹并且增加\_category\_.json文件。
3. push到moi-doc的github repo， vercel会自动进行部署，所作的修改就会更新到网页上。
